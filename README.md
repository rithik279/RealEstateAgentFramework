# Real Estate Auto Message Bot

This project is focused on automatic lead follow-up for a real estate advisory business.

It supports:
- Email
- SMS
- Facebook Messenger

Main use case:
- contact new leads quickly
- push a booking link
- promise 3 personalized homes in 1 day
- reinforce savings and full-service value
- stop automation when a lead replies

MVP browser UI:
- open `/mvp` in the app
- create a Facebook lead with name, email, and phone
- paste your script or talking points
- send the first SMS and continue the conversation as the lead

OpenAI setup:
- add `OPENAI_API_KEY` to `.env`
- default model is `gpt-5-mini`
